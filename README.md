# GrowTive

AI 기반 개인 성장 플랫폼

## Features

- Money Dashboard
- Financial Snapshot System
- Template-based Finance Structure
- Sankey Salary Flow Visualization
- WebSocket Chat (GrowTalk)

## Tech Stack

Backend
- Spring Boot
- MyBatis
- MariaDB

Frontend
- Vanilla JS SPA
- Axios
- Chart.js

## Architecture

Page → Axios → Spring Controller → Service → MyBatis → MariaDB
